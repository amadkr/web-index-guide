# web-index-guide

一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为通用页面集合的存储与展示空间。

## 项目简介

`web-index-guide` 旨在提供一个结构化的目录，用于存放各类独立 HTML 页面。这些页面可能包含文档、工具、演示或实验性内容，彼此之间并无强关联。本仓库的主要目的是方便页面的集中归档与版本管理，同时借助 GitHub Pages 实现简单的静态发布。

## 目录说明

仓库根目录下按页面类型或用途划分若干子目录，每个子目录内包含一个或多个独立的 HTML 文件。建议的目录结构如下：

```
web-index-guide/
├── docs/          # 文档类页面
├── tools/         # 工具类页面
├── demos/         # 演示或示例页面
├── experiments/   # 实验性页面
└── index.html     # 仓库首页（可选）
```

- 每个子目录可包含独立的 `index.html` 作为该目录的入口页。
- 页面之间的链接应尽量使用相对路径，以保持仓库的可移植性。

## 页面归档说明

- 所有页面均为独立的 HTML 文件，可包含内联 CSS 与 JavaScript，不依赖外部资源（除非特别说明）。
- 页面文件命名应简洁明了，避免使用中文或特殊字符。
- 如需引用图片、样式或脚本，建议放置在同目录下的 `assets/` 子目录中。
- 本仓库不限制页面内容主题，但请确保内容合法、不侵犯第三方权益。

## 维护说明

- 欢迎提交 Pull Request 添加新的页面或改进现有页面。
- 提交前请确保页面在主流浏览器中可正常访问。
- 不建议在页面中嵌入跟踪代码、广告或外部服务，以保持页面的轻量与隐私友好。
- 仓库维护者会定期检查页面可用性，但不对页面内容的长期有效性做承诺。
